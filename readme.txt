== Description ==

WP Buttons (formerly <a href="http://dev.linksku.com/sharepost/" title="WP Buttons WordPress Plugin">Share Post</a>) adds sharing buttons to your WordPress blog. Allows your users to easily share your post in one click!
Many unique coding techniques were implemented in WP Buttons for maximum speed and usability. You must try out WP Buttons to see why its much better than other sharing plugins!

<strong>Note: If you are having trouble downloading from WordPress, you can download WP Buttons from <a href="http://code.google.com/p/share-post/downloads/list">Google Code</a></strong>

<strong>Increase traffic</strong>

<strong>Click counts</strong>
Each button keeps track of the number of clicks each button has. Every time someone clicks a button, the count goes up.

WP Buttons reduces the number of HTTP requests by embeding the buttons directly on your blog.

WP Buttons comes with all of the most popular social bookmarking buttons including Like, Tweet, Digg, +1, Delicious, Reddit, LinkedIn, Follow, and much more!

== Frequently Asked Questions ==

= What are social bookmarking buttons? =

Social bookmarking refers to sharing links on social sites like Facebook. It's another term for sharing buttons. Facebook's Like button, Twitter's Tweet buttons, etc are all examples of social bookmarking buttons.

= What makes this plugin better than the other similar plugins? =

WP Buttons is the only plugin of its type that uses the official buttons developed by the social websites. This enables WP Buttons to retreive the click counts (Likes, Tweets, etc) directly from the websites' server.
In addition, WP Buttons uses advanced PHP techniques to dynamically generate different codes for maximum compatibilty with different browsers and operating systems.

= What are the counters? =

The counters included with every button is a real-time (live) count of the number of clicks each button has. Examples are the number of Likes, Tweets, or Diggs.

= How can I customize this plugin? =

This plugin can be easily customized using the many customization settings in `WordPress Admin > Settings > WP Buttons`. You can choose which buttons to display, and in which order are they displayed in.

= How do I display the buttons outside the loop? =

You can put `<?php sharepost_buttons($post_id); ?>` anywhere in your template to display social bookmarking buttons for the provided post ID.
